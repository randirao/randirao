## 🚀 실행 방법

이 포트폴리오는 React + TypeScript + Tailwind CSS로 제작되었습니다.

### Prerequisites

- Node.js 18.x 이상
- npm 또는 yarn

### Installation

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 빌드 결과물 미리보기
npm run preview
```

개발 서버는 기본적으로 `http://localhost:3000`에서 실행됩니다.
